### 计算实时热门商品

每隔5分钟输出最近一小时内点击量最多的前N个商品

数据结构：

    用户ID	     整数类型，加密后的用户ID
    商品ID	     整数类型，加密后的商品ID
    商品类目ID    整数类型，加密后的商品所属类目ID
    行为类型	     字符串，枚举类型，包括(‘pv’, ‘buy’, ‘cart’, ‘fav’)
    时间戳	     行为发生的时间戳，单位秒
